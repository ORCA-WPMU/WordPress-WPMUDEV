## Prerequisites

 - PHP > 5.4
 - composer
 - grunt

## Building WordPress
```Bash
git clone https://github.com/BernieBots/Wordpress-WPMUDEV.git
cd Wordpress-WPMUDEV
npm install
composer install
composer compile
```
